tkinter-cal
===========

A simple calculator using module tkinter
模仿的是小米手机端的界面，练习使用，计算处理部分使用的是eval函数．
初学Tkinter，挺喜欢的．

其中存在的问题:
===============
	*内容显示栏，只能显示一行，对于长数字，显示不全;
	*每次输入完一个式子，必须清空后才可以输入下一个;
	*没有添加菜单栏 ;
	*添加按钮，没有使用循环语句，不符合Python习惯.
